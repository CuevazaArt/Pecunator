# REST Weight Monitor Tool Module

- Runtime tool package: `runtime/modules/tools/rest_weight/`
- Core logger: `runtime/core/rest_usage_log.py`
- API telemetry endpoints:
  - `/api/v1/usage/rest-weight/samples`
  - `/api/v1/usage/rest-weight/events`
  - `/api/v1/usage/rest-weight/report`
- Data store: `runtime/data/rest_usage_samples.sqlite`
- UI monitor dialog: `desktop_shell/lib/main.dart` (`_RestWeightMonitorDialog`)
